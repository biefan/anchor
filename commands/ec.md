# /ec — alias for /anchor

This was the primary slash command in anchor v1.0 - v1.4.x. Starting v1.5.0 the main skill name is `anchor` to match the project brand. `/ec` continues to work as an alias.

**Action:** Load and apply the `anchor` skill (formerly `ec`) — full rules in `~/.claude/skills/anchor/SKILL.md`. Same behavior, just renamed.

If you typed `/ec` out of habit, no need to change — this command does exactly what `/anchor` does.
